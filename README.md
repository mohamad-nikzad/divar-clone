## Getting Started

First, install dependencies:

```bash
npm i
# or
yarn
```

Then, start json-serve:

```bash
npm start:server
# or
yarn start:server
```

And last, open new terminal and start dev server:

```bash
npm dev
# or
yarn dev
```
